Тема: структура html-документа. вибір предметної галузі. робота з посиланнями, таблицями, зображеннями, списками в html-документі.

Мета: придбати практичні навички роботи з HTML-документом, таблицями, зображеннями, посиланнями, списками, формами. Створити шаблон звітного HTML-документом для відображення результатів роботи всіх лабораторних робіт.

Опис предметного середовища:

Наш інтернет-магазин реалізує основні можливості популярних інтернет-магазинів, наприклад перегляд товарів різних категорій, створення замовлення з товарів, що знаходяться в корзині тощо. Цей магазин є навчальним проєктом для того, щоб навчитись основам Front-End та Back-End розробки.

Детальніше про функціональні та нефункціональні вимоги можна прочитати на сторніці Github (посилання).

Висновки:
